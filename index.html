Purpose of the WebSite

The purpose of this website is to practice the subject that I learn in IS-601 class. Subject that I learn and use are listed below.

Linux Commands

    cd = Change Directory mkdir = Make Directory pwd = Show Current Directory ls = Directory Listing cd .. = Move to upper directory cd ~ = Move to home directory

Vi Commands

  • W = Write
  • Q = Quit
  • WQ = Write and Quit
  • vi FILENAME =Start Vi Editor
  • esc key = Exit Insert Mode

Git Commands

  • git init
  • git status
  • git add
  • git commit -m "Task:Update File"
  • git checkout -b branch
