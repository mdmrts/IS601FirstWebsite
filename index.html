Purpose of the WebSite

The purpose of this website is to practice the subject that I learn in IS-601 class. Subject that I learn and use are listed below.

Linux Commands

  1. cd = Change Directory
  2. mkdir = Make Directory
  3. pwd = Show Current Directory
  4. ls = Directory Listing
  5. cd .. = Move to upper directory
  6. cd ~ = Move to home directory
  7. clear=clear screen in front of you

Git Commands

  1. git init= initialize an existing directory as a Git repository
  2. git status=show modified files in working directory, staged for your next commit
  3. git add=add a git URL as an alias
  4. git commit -m =commit your staged content as a new commit snapshot "Task:Update File"
  5. git checkout -b branch
  6. git merge=merge the specified branch’s history into the current one
  7. git config --global user.name “[firstname lastname]”= set a name that is identifiable for credit when review version history
  8. git config --global user.email “[valid-email]”= set an email address that will be associated with each history marker
  9. git push=Transmit local branch commits to the remote repository branch

Vi Commands

  1. w = Write
  2. q = Quit
  3. wq = Write and Quit
  4. vi filename =Start Vi Editor
  5. esc key = Exit Insert Mode
  6. I=insert text before cursor
